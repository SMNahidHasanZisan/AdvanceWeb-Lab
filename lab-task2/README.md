# Lab Task 02: Course Management REST API

This NestJS project extends Lab Task 01 with DTO validation, global pipes, structured JSON responses, and course material upload.

## Run

```bash
npm install
npm run start:dev
```

## Routes

- `GET /course`
- `GET /course/:id`
- `POST /course`
- `PUT /course/:id`
- `PATCH /course/:id`
- `DELETE /course/:id`
- `POST /course/:id/upload`
